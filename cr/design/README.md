## 设计原则

1. **分层解耦** - 每个审核维度独立，可单独启用/禁用
2. **渐进增强** - 从简单规则开始，逐步叠加复杂分析
3. **可配置** - 不同场景选择不同审核维度组合
4. **可度量** - 每条审核结果可追踪、可反馈、可优化
