## 核心设计理念

| 理念                 | 说明                               |
| -------------------- | ---------------------------------- |
| **分层解耦**         | 10 个审核维度独立，可单独启用/禁用 |
| **渐进增强**         | 从简单规则开始，逐步叠加复杂分析   |
| **先个人后团队**     | 先在本地打磨审核质量，再接入 CI    |
| **宁可漏报不可误报** | 宽松起步，确保每条问题都是真问题   |
